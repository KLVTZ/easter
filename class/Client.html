Overview

This class is abstract. This class should not be instantiated directly.

Base class for Rabbitmq client implementations. Acts as an abstract class documenting the api for all implementations.

Constructor Details

# (void) constructor(options = {})

Builds an client for rabbitmq server. All available options are detailed here, but one should only pass only the needed options for the specific implementations.

Parameters:

  • options ( Object )

Options Hash: (host):

  • Ip ( String ) — of machine where rabbitmq server is installed. Defaults to 'localhost'.

Options Hash: (port):

  • Port ( Number ) — where the rabbitmq server is listening. Defaults to 5672 for amqp.

Options Hash: (vhost):

  • Rabbitmq ( String ) — virtual host. Defaults to '/'.

Options Hash: (login):

  • Username ( String ) — to login to the api. Defaults to 'guest'

Options Hash: (password):

  • Password ( String ) — to login to the api. Defaults to 'guest'

This method is abstract.

Publishes a message to the specified queue.

Returns:

  • ( Q.Promise ) — Resolves when the message is published.

# (void) subscribe(queueName, callback, options = {})

This method is abstract.

Subscribes to a queue. Whenever a new message is available the provided callback is executed with that message. Make sure you ack() to consume the message.

Parameters:

  • queueName ( String ) — The queue will be created if it does not exist.
  • callback ( Function ) — function (error, message, ack) {...}
  • options ( Object ) — Options specific to the protocol implementation.
